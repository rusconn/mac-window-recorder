# 開発コマンド

## デバッグ実行

```sh
swift run window-recorder
```

## リリース実行

```sh
swift run -c release window-recorder
```

## デバッグビルド

```sh
swift build
```
## リリースビルド

```sh
swift build -c release
```

## スタッター確認用ウィンドウ表示

```sh
./scripts/smooth-window.sh
```

## キャプチャ結果スタッター確認

```sh
./scripts/verify-stutter.sh <path/to/video> [expected_fps] [skip_seconds]
```

| 引数 | デフォルト | 説明 |
|---|---|---|
| video | - | キャプチャ結果のファイルパス |
| expected_fps | 60 | 期待フレームレート |
| skip_seconds | 1.0 | 開始直後の安定しないフレームを除外する秒数 |

## ffmpegの更新
